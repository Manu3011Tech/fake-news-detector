# 🛡️ Fake News Detection System

A multi-modal AI system for detecting fake news in text, images, and combined content.

## ✨ Features

- **Text Analysis**: Detect fake news articles using NLP
- **Image Analysis**: Identify manipulated or AI-generated images
- **Multi-modal Analysis**: Combined text+image analysis for better accuracy
- **Visualization**: Interactive charts showing confidence scores and analysis breakdown
- **Chatbot Assistant**: Get explanations and verification tips
- **Real-time Feedback**: Instant predictions with detailed reasoning
